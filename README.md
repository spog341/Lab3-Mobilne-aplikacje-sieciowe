Aplikacja po uruchomieniu automatycznie pobiera dane z serwera metodą GET i wyświetla listę postów.
Użytkownik może dodać nowy wpis przez formularz, który wysyła dane metodą POST w formacie JSON.
Aplikacja obsługuje stan ładowania, błędy połączenia oraz wyświetla odpowiedź serwera po poprawnym zapisie.

Podczas testów wykorzystywany był inny endpoint ponieważ ten podany w zadaniu jest niedostępny.
